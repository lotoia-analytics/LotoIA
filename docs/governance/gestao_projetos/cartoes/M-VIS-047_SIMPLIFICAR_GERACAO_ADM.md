# M-VIS-047 — Simplificar Página de Geração ADM CORE_002

| Campo | Valor |
|-------|-------|
| Missão | M-VIS-047 |
| Tipo | Simplificação visual / operacional |
| Build ADM | `institutional-adm-runtime-v22` |
| Pré-requisito | M-VIS-046, M-ML-045, M-GER-044 |

## Problema

A página **Gerador ADM CORE_002** exibia status constitucional completo, avisos repetidos e detalhes técnicos antes dos controles de geração — carga visual excessiva para o operador.

## Correção

- Cabeçalho compacto: geração, ML, Lei 15A inoperante, PostgreSQL
- Bloco operacional: quantidade, formato `15 dezenas — CORE_002 soberano`, estratégia `CORE_002 + ML supervisionado`, botão gerar
- Resultado resumido pós-geração
- Expansores: Detalhes de governança | Detalhes técnicos | Leitura Lei 15 / 6 Bases
- Textos proibidos (16D–23D, expansão auditada, Lei 15A operacional) ausentes do fluxo principal

## Veredicto alvo
